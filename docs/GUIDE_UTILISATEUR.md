# Guide Utilisateur NovaClub

Guide complet pour utiliser NovaClub au quotidien.

## Connexion

1. Ouvrir le navigateur
2. Aller à l'adresse de votre club (ex: http://192.168.1.8:3000)
3. Entrer votre email et mot de passe
4. Cliquer sur "Se connecter"

## Tableau de Bord

Le tableau de bord affiche:
- Nombre total d'adhérents
- Adhérents actifs
- Paiements en attente
- Revenus du mois

### Navigation

Utilisez le menu en haut pour accéder aux différentes sections:
- **Tableau de bord**: Vue d'ensemble
- **Adhérents**: Gestion des membres
- **Paiements**: Enregistrement des cotisations
- **Déconnexion**: Se déconnecter

## Gestion des Adhérents

### Ajouter un adhérent

1. Aller dans "Adhérents"
2. Cliquer sur "Nouvel adhérent"
3. Remplir le formulaire:
   - **Informations personnelles**: Nom, prénom, date de naissance, sexe
   - **Contact**: Téléphone, email, adresse
   - **Parent**: Pour les mineurs, renseigner le nom et contact du parent
   - **Judo**: Catégorie, discipline, ceinture
   - **Cotisation**: Montant mensuel
4. Cliquer sur "Enregistrer"

### Modifier un adhérent

1. Dans la liste des adhérents, cliquer sur l'adhérent
2. Modifier les informations
3. Cliquer sur "Enregistrer"

### Statuts des adhérents

- **Actif**: Adhérent à jour de ses cotisations
- **En attente**: Inscription en cours
- **Suspendu**: Cotisations en retard
- **Inactif**: N'est plus membre du club

### Rechercher un adhérent

Utilisez la barre de recherche en haut de la liste pour trouver rapidement un adhérent.

## Gestion des Paiements

### Enregistrer un paiement

1. Aller dans "Paiements"
2. Cliquer sur "Nouveau paiement"
3. Sélectionner l'adhérent
4. Choisir le type:
   - **Cotisation mensuelle**: Paiement mensuel régulier
   - **Inscription**: Frais d'inscription annuelle
   - **Équipement**: Achat de judogi, ceinture, etc.
   - **Licence**: Renouvellement de licence
   - **Autre**: Autres paiements
5. Entrer le montant
6. Choisir la méthode:
   - **Espèces**: Paiement cash
   - **Mobile Money**: Orange Money, Wave, etc.
   - **Virement**: Virement bancaire
7. Sélectionner la date
8. Pour les cotisations, sélectionner le mois concerné
9. Ajouter des notes si nécessaire
10. Cliquer sur "Enregistrer"

### Consulter l'historique

La liste des paiements affiche:
- Date du paiement
- Nom de l'adhérent
- Type de paiement
- Montant
- Méthode
- Statut

### Filtrer les paiements

Utilisez les filtres pour:
- Voir les paiements d'un mois précis
- Filtrer par type
- Filtrer par statut

## Fonctionnement Hors Ligne

### Mode Hors Ligne

Quand Internet n'est pas disponible:
- Un indicateur apparaît en bas à droite
- Vous pouvez continuer à travailler normalement
- Toutes les modifications sont enregistrées localement
- Quand Internet revient, tout se synchronise automatiquement

### Synchronisation

La synchronisation se fait:
- Automatiquement toutes les 30 secondes
- Immédiatement lors de la reconnexion
- En arrière-plan sans interruption

Vous verrez un indicateur de synchronisation pendant le processus.

## Application Mobile Coach

### Installation

1. Télécharger le fichier APK sur le téléphone
2. Autoriser l'installation depuis des sources inconnues
3. Installer l'application
4. Ouvrir NovaClub Coach

### Configuration

1. Au premier démarrage, entrer l'adresse IP du serveur
   - Exemple: http://192.168.1.8:8000
2. Se connecter avec les identifiants coach

### Fonctionnalités

#### Voir les adhérents
- Liste complète des judokas du club
- Recherche rapide
- Détails de chaque adhérent

#### Marquer les présences
1. Aller dans "Présences"
2. Sélectionner la date (par défaut aujourd'hui)
3. Cocher les adhérents présents
4. Enregistrer

#### Enregistrer un paiement
1. Aller dans "Paiements"
2. Sélectionner l'adhérent
3. Entrer le montant
4. Choisir le type
5. Enregistrer

#### Ajouter un adhérent
Même formulaire que sur le web, adapté pour mobile.

### Mode Hors Ligne Mobile

L'application mobile fonctionne aussi hors ligne:
- Toutes les données sont en local
- Modifications enregistrées localement
- Synchronisation automatique avec le serveur

## Bonnes Pratiques

### Saisie des données

1. **Cohérence**: Utilisez toujours le même format
   - Téléphones: +221 XX XXX XX XX
   - Emails: minuscules
   - Noms: Première lettre majuscule

2. **Complet**: Remplissez tous les champs importants
   - Contact parent pour les mineurs
   - Email pour les notifications
   - Date de naissance exacte

3. **À jour**: Mettez à jour régulièrement
   - Changements d'adresse
   - Nouveaux numéros
   - Passages de grade

### Gestion des cotisations

1. **Enregistrez immédiatement**: Dès réception du paiement
2. **Vérifiez le mois**: Assurez-vous de sélectionner le bon mois
3. **Notes**: Ajoutez des notes pour les cas particuliers
4. **Reçus**: Conservez une trace écrite

### Sécurité

1. **Déconnexion**: Déconnectez-vous après usage
2. **Mot de passe**: Changez-le régulièrement
3. **Confidentialité**: Ne partagez pas les données personnelles
4. **Sauvegarde**: Faites des sauvegardes régulières

## Questions Fréquentes

### Que faire en cas de coupure de courant?

NovaClub enregistre automatiquement. Au redémarrage, toutes vos données seront là.

### Comment ajouter un nouvel utilisateur?

Seul l'administrateur peut ajouter des utilisateurs dans "Paramètres" > "Utilisateurs".

### Puis-je modifier un paiement déjà enregistré?

Non, mais vous pouvez ajouter une note ou créer un avoir.

### Comment exporter les données?

Dans "Comptabilité" > "Exports", vous pouvez exporter en Excel ou PDF.

### L'application est lente, que faire?

1. Vérifier la connexion réseau
2. Fermer les autres applications
3. Redémarrer le navigateur
4. Contacter le support si le problème persiste

### Comment voir qui est en retard?

Dans le tableau de bord, le nombre de "Paiements en attente" indique les adhérents en retard.

### Puis-je utiliser NovaClub sur plusieurs ordinateurs?

Oui! Tous les ordinateurs du réseau peuvent accéder à l'application en même temps.

### Les données sont-elles sécurisées?

Oui, toutes les données sont stockées localement et protégées par mot de passe.
